# helpme docs

This is the generic MkDocs tree used by `helpme`.

## Included

- `helpme shell/` documents the terminal editing flow
- `javascripts/path-copy.js` and `stylesheets/path-copy.css` add copyable rendered file paths

## Start

```bash
docker compose up -d docs
```

Then open:

```text
http://127.0.0.1:18101/
```
